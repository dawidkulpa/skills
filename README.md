# Skills

I'm building a collection of specialized instruction sets for AI agents. Each skill encodes best practices, workflows, and domain knowledge for specific tasks—from debugging and code review to game design and competitive analysis.

This repository contains portable, agent-agnostic skills you can use with Claude, Gemini, Cursor, or any other AI platform that supports custom instructions or MCP-style skill loading.

## How to Use

1. **Browse available skills** in the `skills/` directory. Each skill folder contains a `SKILL.md` file with detailed instructions and workflows.
2. **Copy the skill folder** into your agent's skill directory (location depends on your platform).
3. **Load the skill** when you need it. The `SKILL.md` file is the entry point—your agent will read it and follow the instructions.

That's it. No installation commands, no configuration files. Just copy and use.
